# 历史年表网站

一个交互式历史年表网站，支持时间轴滑动浏览历史事件，点击人物可查看详细档案。

## 功能特性

### 核心功能

- **时间轴浏览** - 横向滚动时间轴，展示历史事件
- **事件卡片** - 显示事件时间、标题、描述、分类和相关人物
- **人物档案** - 点击人物名称查看详细档案（弹窗形式）
- **导航筛选** - 按事件类型筛选（政治、文化、科技、军事）
- **搜索功能** - 搜索事件或人物

### 高级功能

- **标签系统** - 事件可添加自定义标签
- **收藏功能** - 收藏感兴趣的事件、人物、年份
- **分享功能** - 分享特定事件或人物
- **时间轴缩放** - 调整时间轴显示密度（50%-200%）
- **多语言支持** - 中英文切换
- **离线支持** - 使用 Service Worker 缓存数据
- **登录验证** - 除滑动查看外的功能需登录验证

## 使用方法

### 1. 打开网站

直接在浏览器中打开 `timeline.html` 文件即可使用。

### 2. 浏览时间轴

使用鼠标滚轮或触摸板横向滚动时间轴，点击事件卡片可查看详情，点击人物名称可查看人物档案。

### 3. 筛选和搜索

使用顶部筛选器按事件类型筛选，在搜索框中输入关键词搜索事件或人物。

### 4. 缩放时间轴

点击 `+` 按钮放大时间轴，点击 `-` 按钮缩小时间轴，缩放范围：50% - 200%

### 5. 收藏和分享

点击事件卡片或人物档案中的"收藏"按钮收藏内容，点击"分享"按钮分享内容（支持原生分享或复制到剪贴板）。

### 6. 多语言切换

点击"中文/English"按钮切换语言。

### 7. 登录

点击"登录"按钮打开登录弹窗，输入用户名和密码登录，登录后可使用收藏、分享等功能。

```javascript
{
    id: "event_001",
    year: "公元前221年",
    title: "秦始皇统一六国",
    description: "秦灭六国，建立中国历史上第一个统一的多民族国家",
    category: "political",
    characters: [
```

## 技术栈

- **HTML5** - 页面结构
- **CSS3** - 样式和动画
- **JavaScript (ES6+)** - 交互逻辑
- **Service Worker** - 离线支持
- **LocalStorage** - 本地数据存储

- Chrome 60+
- Firefox 55+
- Safari 11+
- Edge 79+

## 自定义数据

### 添加新事件

在 `data/events.js` 中添加新的事件对象到 `historicalEvents` 数组。

### 添加新人物

在 `data/characters.js` 中添加新的人物对象到 `historicalCharacters` 数组。

### 修改样式

在 `timeline.css` 中修改样式变量和类样式。

## 注意事项

1. **离线支持** - 首次访问需要联网，之后可离线使用
2. **数据存储** - 收藏和登录信息存储在浏览器 LocalStorage 中
3. **Service Worker** - 需要通过 HTTP/HTTPS 协议访问才能正常工作
4. **响应式设计** - 支持桌面端和移动端，建议在桌面端使用以获得最佳体验

## 开发说明

### 添加新功能

1. 在 `timeline.js` 的 `TimelineApp` 对象中添加新方法
2. 在 `timeline.html` 中添加相应的 UI 元素
3. 在 `timeline.css` 中添加样式

### 修改数据结构

确保同时更新 `data/events.js` 和 `data/characters.js` 中的数据，保持数据一致性。
